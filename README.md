# Welcome To Project Leaderboards!

We are Project Leaderboards! Our goal is to create a unique and immersive gaming experienced for users of all backgrounds! 

Our website is a collection of interconnected games, allowing users to blend classic game titles with modern gaming experiences!

## Installation
1. Install [Node.js](https://nodejs.org)

2. Clone the repository to a directory on your local machine.

3. Navigate to the repository directory in your Terminal/Command Line, using File > Open Folder..., and then going to Terminal > New Terminal, which will create a terminal prompt in the directory at the bottom of the window) and install the dependencies with the following command: ```npm i```

## Getting Started
First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Next, open [http://localhost:3000](http://localhost:3000) with your browser to see the result!
